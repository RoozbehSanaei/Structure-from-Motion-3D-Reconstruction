# TempleRing SfM (Python)

Main entrypoint:
- `src/templering_sfm.py`

Install:
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Run:
```bash
python src/templering_sfm.py --help
python src/templering_sfm.py --data <templering_root> --out outputs --max-frames 12
```
